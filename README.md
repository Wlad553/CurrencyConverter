# CurrencyConverter
Application for converting currencies, which uses forex rates

# Branch
Current branch - main

# Features
* iPadOS support
* Currency converting
* 52 Currencies to convert
* Works online/offline
* App updates and stores currency rates once an hour in background
* If user didn't open the app for more than 24 hours, then app updates rates once a day
* Favourite currencies list saving
* Favourite currencies list order saving after being moved 
* Favourite currencies list consists of max 4 currencies, which allows to convert simoultaniously to 3 other currencies
* Share rates feature

# Stack
* Swift
* UIKit
* MVC
* Core Data
* BackgroundTasks
* URLSession
* UI Storyboard/Programmatically (Constraints)
* DI
* Unit and UI Tests
